Bataviegroep Rental-portal website

The Bataviagroep Rental-portal is a lightweight easy to use portal website. Its easy to intergrade with in thirdparty systems. My role in this project was to do the design from scratch of. And also all the Front-end. What i used was HTML CSS3 and jQuery i validated the code via w3c. ANd its cross browser tested

The full responsive Portal

The full responsive home/start page is a simple and clear page. It contains two main options the user can choice to go to buy or rent. There is a main menu and a footer. On moblie the big footer will disappear to safe space.

The object overviw page

The offer overview page contains filters wich directly changes the content. The filters disappear on mobile but with a single click on the button the are back full screen.

Object detail page.

Object detail page. Here i used an accordion to safe space. Wich is realy easy to use. There is a two- division made between two tabs. Overview and photos and maps. Each contains there own exclusive content

Full responsive

The form's are desktop and mobile friendly to use. The website is build with the Component Based Development (CBD) method. That means that every single thing is an component and can be used every where else on the website.
